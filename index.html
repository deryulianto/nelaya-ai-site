Fitur Utama

🤖 AI Inference (FGI)

Prediksi Fish Growth Index dari parameter laut (Suhu, Salinitas, Klorofil) beserta kategori Low/Medium/High.

📈 Evaluator & Insight

Logging otomatis, statistik, tren, dan evaluasi performa model untuk pengambilan keputusan.

🌐 Visualizer 3D

Plot 3D interaktif hubungan Suhu–Salinitas–Klorofil dan pengaruhnya terhadap FGI.

Integrasi Data

Platform mendukung integrasi dengan penyedia data oseanografi satelit & in-situ. Kompatibel dengan layanan berbasis standar terbuka (OGC/ERDDAP/NetCDF) serta API pihak ketiga.

🛰️ Satelit: SST, Klorofil, Arus Permukaan
🌊 Model & Reanalisis: variabel fisik & biogeokimia
🧪 In-situ: ARGO, buoy, sensor farm
